WooDurcoin  is a payment gateway plugin for WooCommerce. It integrates payment using Durcoin  assets via Waves.Exchange.

== Description ==

= Plugin Description =

The plugin displays a payment form with the amount in the selected asset on the checkout page.
The amount in the site currency is automatically converted into the amount in the asset. Adds the ability to use the Durcoin  asset as the main currency of the site.

== Installation ==

= Minimum Requirements =

* WooCommerce >= 3.6.4
Further minimum requirements are determined by the requirements for WooCommerce:
* PHP 5.6 or greater
* PHP GMP extension
* MySQL 5.6 or greater
* WordPress 5.8+

Furthermore, you need a Durcoin .Exchange account on (https://waves.exchange).

= Manual Installation =

Download the plugin from here. Unpack the archive and load the folder via FTP into the directory `wp-content\plugins` of your WordPress installation. Go to *Plugins => Installed plugins* and click *Activate* on *WooDurcoin *.
